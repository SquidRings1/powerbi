## 🔍 Troubleshooting

### Issue: "ModuleNotFoundError: No module named 'pandas'"

**Solution**:
```bash
pip install pandas numpy
```

### Issue: Template file not created

**Solution**:
1. Verify Python is running without errors
2. Check write permissions in project folder
3. Ensure zipfile module is available (built-in)

### Issue: CSV files not loading in Power BI

**Solution**:
1. Verify files exist: `ls data/` (Mac/Linux) or `dir data` (Windows)
2. Check file permissions
3. Try loading one CSV at a time
4. Use Transform Data instead of Load for complex tables

### Issue: Dashboard shows "No data"

**Solution**:
1. Run refresh: Ctrl+R (Windows) or Cmd+R (Mac)
2. Check relationships (Model tab → Manage Relationships)
3. Verify data loaded: Data tab → View table contents
4. Check measure formulas in DAX editor

### Issue: Colors not displaying correctly

**Solution**:
1. Close and reopen Power BI
2. Theme may need manual adjustment
3. Check View → Theme settings
4. Re-apply dark theme from ribbon
